# 🎓 Tuition Class Management System

A modern, responsive web application for managing tuition classes efficiently. 
Built with React 19, Vite, and Tailwind CSS.

## ✨ Features
- 👨‍🎓 Student & Parent Dashboard
- 👨‍🏫 Tutor Portal
- 🛡️ Admin Dashboard
- 📅 Attendance Management
- 💰 Fee & Enrollment Tracking
- 📚 Subject Management
- 💬 Messaging System
- 🔐 Role-based Login (Admin / Tutor / Student / Parent)

## 🛠️ Tech Stack
- **Frontend:** React 19, React Router DOM
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Build Tool:** Vite 7

## 🚀 Getting Started
cd frontend
npm install
npm run dev
